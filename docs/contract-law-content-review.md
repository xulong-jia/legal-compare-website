## 总体结论

未发现阻断问题。12 张卡片整体保持了学习型、比较法导向和非法律意见的表达方式，frontmatter 的 `summary`、`tags`、`difficulty` 与正文主题基本一致，四阶段学习路径衔接自然。

主要需要后续处理的是结构和术语统一：前 5 张卡片仍使用早期栏目结构，例如“中国法规则”“大陆法系背景”“英美法系规则”“法条 / 判例 / 来源”“考试提示”“论文角度”“实务提示”；后 7 张卡片使用较新的“核心问题 / 中国法 / 大陆法系 / 普通法系 / 主要差异 / 学习提示 / 示例 / 常见误区 / 延伸问题”结构。建议后续统一为新结构。

## 术语一致性检查

- “中国法”使用较稳定，但前 5 张多写作“中国法规则”，后 7 张写作“中国法”。
- “大陆法系”使用较稳定，但前 5 张多写作“大陆法系背景”，后 7 张写作“大陆法系”。
- “普通法系”与“英美法系”存在用词差异：前 5 张 frontmatter `systems` 使用 `common-law`，正文多写“英美法系规则”或“英美合同法”；后 7 张多写“普通法系”。
- “合同成立”“合同效力”“违约责任”“损害赔偿”“不可抗力”“履行障碍”“格式条款”“对价”等核心术语整体一致。
- “force majeure clause”“frustration”“expectation damages”“remoteness”“mitigation”等英文术语保留合理，但后续可考虑在首次出现时统一补充中文说明。

## 各卡片审校意见

### 要约邀请

内容准确性和学习定位基本稳定。主要问题是栏目结构仍是早期版本，且“英美法系规则”建议后续统一为“普通法系”。法条和案例来源较具体，后续扩写前建议人工法律复核。

### 要约

summary 提到“撤回规则”，正文已覆盖撤回、撤销和失效，前后基本一致。建议后续统一栏目结构，并将“英美法系规则”改为“普通法系”以保持术语一致。

### 承诺

内容覆盖承诺、迟到承诺和 postal rule，结构清楚。建议后续统一栏目结构，并把“考试提示 / 论文角度 / 实务提示”转换为“学习提示 / 示例 / 常见误区 / 延伸问题”。

### 对价

对价与中国法、大陆法系的非对应关系表达较克制，没有明显把对价误写为价款。建议后续补一个更具体的“示例”栏目，帮助初学者区分 sufficient consideration 与 adequate consideration。

### 违约责任

对效率违约的表达已经限定为论文讨论视角，没有写成规则结论。建议后续统一栏目结构，并注意与 `damages` 卡片减少重复：违约责任卡片可更强调责任方式全貌，损害赔偿卡片集中讨论赔偿范围。

### 合同解释

结构、术语和学习提示较成熟，适合作为后续统一模板。未发现明显法律意见风险。

### 格式条款

纳入、解释和效力控制三层结构清楚，summary 与正文一致。个别表述如“都会设置一定的纳入、解释或效力控制规则”略概括，后续可改为“通常会设置”以继续弱化绝对感。

### 合同效力

成立、效力、可执行性的区分清楚。普通法中 `enforceability` 与大陆法系效力分类的关系表达克制，未发现明显混淆。

### 合同解除

“解除不是合同作废”的提示有效。可在后续修订中进一步区分 termination、rescission、frustration 等英文术语，避免初学者把所有“退出合同”都理解为同一制度。

### 损害赔偿

内容与违约责任卡片有合理衔接。后续可把“惩罚性违约金限制”表述再核对一次，避免初学者误读为普通法和中国法都采用相同的概念路径；该点建议人工法律复核。

### 不可抗力

作为进阶专题定位合理，未把不可抗力写成当然免责。`force majeure clause`、`frustration`、`impossibility / impracticability` 的区分方向正确，后续扩写时建议补充“不同普通法法域差异较大”的限定。

### 合同成立的特殊场景

与第一阶段形成自然呼应，适合作为专题收尾。内容没有写成具体法律意见。后续可考虑在“示例”中更明确区分平台展示、下单、系统确认三个节点。
